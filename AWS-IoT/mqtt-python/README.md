#### 실행 방법

1. 가상 환경 세팅

    ```bash
    $ python -m venv <your_venv_name>
    ```

2. 가상환경 시작

    ```bash
    $  <your_venv_name>/Scripts/Activate.ps1
    ```

3. 라이브러리 다운

    ```bash
    $ pip install -r requirements.txt
    ```

4. 파이썬 RUN

    ```bash
    $  python /src/index.py
    ```
